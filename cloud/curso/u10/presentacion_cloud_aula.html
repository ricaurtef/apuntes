Cloud computing en el aula

Alberto Molina Coballes / @alberto_molina y José Domingo Muñoz Rodríguez / @Pledin_JD

Theme by reveal.js

¿Objetivo o herramienta?

  • La creación, despliegue y utilización de aplicaciones nativas de cloud debe ser en sí mismo un objetivo para las enseñanzas profesionales de informática.
  • El uso, instalación, configuración y administración de plataformas de gestión de cloud es cada vez más importante en los perfiles profesionales de TIC
  • Las tecnologías de cloud computing, particularmente IaaS, proporcionan un banco de pruebas excepcional para aprendizaje basado en la práctica, tal como debe ser la formación profesional

Desarrollo de Aplicaciones Web (DAW)

  • Plantear la inclusión paulatina de desarrollo de aplicación nativas para la nube
  • Utilización de "publicación desacoplada" para separar el contenido dinámico de un sitio del contenido estático pesado mediante Content Delivery Networks (CDN) y almacenamiento de objetos
  • Aplicaciones clásicas en PaaS:
    • Despliegue de aplicaciones
    • Sistemas de control de versiones
    • Integración continua
    • Metodologías ágiles

Desarrollo de Aplicaciones Multiplataforma (DAM)

  • Quizás el módulo menos afectado por el uso de tecnologías de cloud computing
  • Aplicaciones móviles:
    • APIs web
    • Almacenamiento de objetos
  • Por los requisitos que tienen, no hay gran diferencia entre utilizar máquinas virtuales y un entorno de instancias en cloud

Administración de Sistemas Informáticos en Red (ASIR)

  • Muy importante conocer y utilizar IaaS, tanto pública como privada
  • Metodologías relacionadas:
    • APIs web
    • Sistemas de control de versiones
    • Orquestación
    • DevOps
  • Un cloud de infraestructura ofrece también un extraordinario laboratorio de prácticas para sistemas, redes o cualquier disciplina relacionada con la administración de sistemas

Sistemas Microinformáticos y Redes (SMR)

  • En algunos módulos profesionales, principalmente de 2º, es muy útil disponer de IaaS como laboratorio de prácticas
  • Sería adecuado ir integrando paulatinamente algunos conceptos o formas de trabajar en cloud computing en el currículo
  • Conocer y utilizar PaaS como opción de despliegue de aplicaciones web
